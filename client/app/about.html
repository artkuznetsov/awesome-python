This website grew from a pet project listing useful Python libraries and frameworks. While pypi is a great resource, it requires packages to be published, and in some cases I wanted to save repos that are just code or perhaps reference Jupyter notebooks.

If you see a useful repo missing, please create an issue on GitHub suggesting it! Data for all the libraries is available for you to use: github_data.json or github_data.csv. The crawler application and Terraform to deploy to AWS is available on Github
